# .NET MAUI Timer & Scoreboard App

Cross-platform .NET MAUI application for managing game timers and scoreboards.

## Features
- Multi-page user interface
- Timer and scoreboard functionality
- Game history view
- Player statistics (goals and assists)
- Navigation between views using MAUI navigation

## Tech stack
- .NET MAUI
- C#
- XAML

## Purpose
This project was developed as a learning project to practice:
- .NET MAUI application structure
- UI navigation and state handling
- Separation of UI and application logic
- Working with collections and data persistence

## Status
Functional learning project. Not intended for production use.
